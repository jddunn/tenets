    tenets.core Package¶

    Core functionality and algorithms for Tenets.

    Core subsystem of Tenets.

    This package aggregates core functionality such as analysis, distillation, ranking, sessions, and related utilities.

    It exposes a stable import path for documentation and users: - tenets.core.analysis - tenets.core.ranking - tenets.core.session - tenets.core.instiller - tenets.core.git - tenets.core.summarizer

    Subpackages¶

    • analysis - Code analysis engines for multiple languages
    • distiller - Context distillation and aggregation
    • examiner - Code examination and metrics
    • git - Git integration and analysis
    • instiller - Tenet and session injection
    • momentum - Development momentum tracking
    • nlp - Natural language processing utilities
    • prompt - Prompt parsing and analysis
    • ranking - File ranking and relevance scoring
    • reporting - Report generation
    • session - Session state management
    • summarizer - Content summarization
