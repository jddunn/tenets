# Installation

## Requirements

- Python 3.9 or higher
- Git

## Install from PyPI

```bash
pip install tenets
```

## Install from Source

```bash
git clone https://github.com/yourusername/tenets.git
cd tenets
pip install -e .
```

## Verify Installation

```bash
tenets --version
```
